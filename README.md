# <p align="center">  marCATplace </p>

<span style="font-size: 16px"><u>Описание</u><br></span>
Симулятор сотрудника ПВЗ с мемными котами <br><br>

<span style="font-size: 16px"><u>Как играть?</u><br></span>

1. **Начало игры**  
   Нажмите на кнопку **Плей** в центре главного экрана.
   ![Кнопка Плей](images/1.png)

2. **Встреча гостя**  
   К вам на стойку ПВЗ придет кот и покажет свой QR-код на экране телефона. Нажмите прямо на этот **QR-код**.
   ![Кот с QR-кодом](images/2.png)

3. **Получение номера**  
   Запомните **число (номер посылки)**, которое отобразится на экране после сканирования.
   ![Номер посылки](images/3.png)

4. **Переход на склад**  
   Перейдите в зону склада с коробками.
   ![Экран склада](images/4.png)

5. **Поиск посылки**  
   Найдите коробку с числом, которое вы только что запомнили, и **нажмите на неё**.
   ![Выбор коробки](images/5.png)

6. **Возвращение**  
   Вернитесь на пункт выдачи (ПВЗ), нажав на **маленькую стрелку** в правом нижнем углу экрана.
   ![Стрелка возврата](images/6.png)

7. **Выдача заказа**  
   Нажмите на кота, чтобы **отдать посылку коту**.
   ![Выдача посылки](images/step7_done.png)
